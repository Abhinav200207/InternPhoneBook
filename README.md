- Fork the repository.

- Clone the repo
   ```sh
   git clone https://github.com/<YOUR_USERNAME>/InternPhoneBook.git
   ```

- Open two terminal/cmd in the `InternPhoneBook` folder

- In one terminal enter
   ```sh
   cd backend
   npm install
   node index.js
   ```

- In second terminal enter
  ```sh
  cd frontend
  npm install
  npm start
  ```
